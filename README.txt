Description:
------------
We have submitted 4 MapReduce job for this code drop. These MapReduce jobs give us useful information for our analysis. The MapReduce jos are as follows:-

JourneyFrequency : This map reduce job get the journey frequency between [start_station_id-stop_station_id].

StartStationFrequency : This map reduce job give the frequency of journeys made from a given start station.

StopStationFrequency : This map reduce job give the frequency of journeys made till a given stop station.

AverageJourneyLength : This map reduce job outputs the average journey time for every startid-stopid combination.
